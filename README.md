# FPGA-Based-Pulse-Height-Analyzer-for-Particle-Spectroscopy
Modular Verilog design for 40MHz LHC data rates. Features a 3-stage pipeline with a 2-cycle latency and 21.8ns WNS. Optimized for efficiency using only 7 LUTs and 92mW. Includes 250ns Veto logic to prevent event pile-up, ensuring 100% noise rejection in high-rate detector environments.
